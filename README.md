# File Selector Table React Component

## Getting Started

### 1. Clone the Repo

```bash
git clone https://github.com/ryan0122/file-table.git
cd file-select-table
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the App locally

```bash
npm start
```

### 4. Build for Production

```bash
npm run build
```

### 5. Run tests
```bash
npm test
```
